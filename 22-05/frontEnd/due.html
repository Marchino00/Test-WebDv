2- Cos'è CSS quali sono le sue caratteristiche
CSS (Cascading Style Sheets) è un linguaggio di stile utilizzato per definire l'aspetto e la formattazione di un documento HTML. Alcune delle caratteristiche principali di CSS includono: - Separazione tra struttura e presentazione: CSS permette di separare la struttura di una pagina web dal suo aspetto visivo, consentendo di definire lo stile e la formattazione in un file CSS separato. - Selezionatori e regole: CSS utilizza selezionatori per individuare gli elementi HTML a cui applicare le regole di stile. Le regole di stile specificano come gli elementi selezionati dovrebbero essere presentati sullo schermo. - Cascading e specificità: CSS supporta la cascata delle regole, consentendo di specificare diversi livelli di priorità per le regole di stile in base alla loro posizione nel codice e alla specificità dei selezionatori utilizzati. - Layout e posizionamento: CSS offre una vasta gamma di proprietà per gestire il layout e il posizionamento degli elementi sulla pagina, come la dimensione, il posizionamento, il flusso del testo e altro ancora. - Supporto per la responsività: CSS permette di creare layout responsivi, adattando l'aspetto di una pagina web in base alle dimensioni dello schermo o al dispositivo utilizzato.
