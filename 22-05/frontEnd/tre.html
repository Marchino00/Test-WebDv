3- cos'è HTML quali sono le sue caratteristiche
HTML (HyperText Markup Language) è il linguaggio di markup standard utilizzato per la creazione di pagine web. Le sue caratteristiche principali sono: - È un linguaggio di markup: HTML definisce la struttura e il contenuto di una pagina web. Utilizza tag e attributi per definire gli elementi e le proprietà di visualizzazione. - Struttura della pagina web: HTML definisce la struttura di una pagina web utilizzando elementi come intestazioni, paragrafi, tabelle, elenchi e altro ancora. Permettendo agli agenti utente di interpretare correttamente il contenuto della pagina. - Collegamenti ipertestuali: HTML permette di creare collegamenti tra diverse pagine web utilizzando l'elemento: "a". Questo consente agli utenti di navigare attraverso i diversi documenti web.
