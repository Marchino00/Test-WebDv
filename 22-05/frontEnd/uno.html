1- Cos'è Js quali sono le sue caratteristiche

JavaScript (JS) è un linguaggio di programmazione ad alto livello, interpretato e orientato agli oggetti. È principalmente utilizzato per lo sviluppo di applicazioni web interattive e dinamiche. Alcune delle caratteristiche principali di JavaScript includono:

- Interattività: JavaScript consente di rendere le pagine web interattive, consentendo agli utenti di interagire con gli elementi della pagina e rispondere a eventi come clic, input dell'utente e altro ancora.

- Manipolazione del DOM: JavaScript consente di modificare il Document Object Model (DOM), ovvero la struttura di un documento HTML, permettendo di aggiungere, rimuovere o modificare elementi e contenuti della pagina in modo dinamico.

- Gestione degli eventi: JavaScript fornisce un sistema di gestione degli eventi che consente di ascoltare e rispondere a eventi come clic del mouse, pressione di tasti e altro ancora.

- Supporto per oggetti e prototipi: JavaScript utilizza un modello di programmazione orientato agli oggetti basato su prototipi, consentendo la creazione e l'ereditarietà di oggetti.

- Ampia compatibilità: JavaScript è supportato dalla maggior parte dei browser moderni, consentendo l'esecuzione di script JavaScript direttamente all'interno di una pagina web.
